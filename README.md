# makeCutawayForGMT
By using this program, from the result file of [FEMTIC](https://github.com/yoshiya-usui/femtic.git), you can make a file needed to draw a slice of a resistivity structure by GMT. 

When GMT v4 or lower is used, please set 'isGMT5OrHigher = false' in 'CommonParameters.h'.
By default, 'isGMT5OrHigher = true' and output files conform to GMT v5 or higher.
